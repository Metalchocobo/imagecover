Imagecover jQuery Plugin

Original Image:



Html Markup:

<div class="containerClass">
  <img src="img/monkey.jpg">
</div>

Use:

$('.containerClass').imagecover();

Advanced Use:

$('.containerClass').imagecover({
  useOnce:false, //if true, he doesn't monitor the conges of the container
  css2:false, //if true, he force css2 fallback and he ignore css3 features
  throttle:200 //interval time to check container changes
});

Requires: imagesLoaded.js - https://github.com/desandro/imagesloaded

Raccomanded (to enable css3 feature): modernizr.js - http://modernizr.com/
